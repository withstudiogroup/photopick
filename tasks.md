# 페이지 리디자인 작업 계획

## 현재 상황
- ✅ 메인 페이지 완성
- 🔄 검색, 스튜디오 상세, 예약 페이지 수정 필요

## 작업 목록

### 1. 검색 페이지 (search/page.tsx) 개선
- [ ] 헤더 영역 디자인 개선 (메인 페이지 스타일 적용)
- [ ] 필터 사이드바 UI 개선
- [ ] 검색 결과 카드 디자인 개선
- [ ] 페이지네이션 스타일 개선
- [ ] 모바일 반응형 개선

### 2. 스튜디오 상세 페이지 (studio/[id]/page.tsx) 개선
- [ ] 이미지 갤러리 디자인 개선
- [ ] 스튜디오 정보 카드 개선
- [ ] 상품 선택 섹션 개선
- [ ] 리뷰 섹션 개선
- [ ] 사이드바 쿠폰/예약 영역 개선

### 3. 예약 페이지 (booking/page.tsx) 개선
- [ ] 스텝 진행바 디자인 개선
- [ ] 캘린더 컴포넌트 스타일 개선
- [ ] 옵션 선택 카드 개선
- [ ] 결제 화면 개선
- [ ] 완료 화면 개선

### 4. 추가 페이지 생성 (선택)
- [ ] 마이페이지 생성
- [ ] 로그인/회원가입 페이지 생성
- [ ] 이벤트 상세 페이지 생성

## 디자인 원칙 (메인 페이지 기준)
- **폰트**: Gowun Batang (타이틀), 본고딕 (본문)
- **색상**: Gold(#C9A962), Dark(#2D2D2D), Cream(#F5F0E8)
- **컴포넌트**: 부드러운 섀도우, 미니멀한 라인, 적절한 여백
- **인터랙션**: Framer Motion 애니메이션

## 우선순위
1. 검색 페이지 → 사용자 flow 핵심
2. 스튜디오 상세 → 전환율 중요
3. 예약 페이지 → 결제 전 단계
