# PhotoPick - 사진 촬영 스튜디오 예약 플랫폼 PRD

## 1. 제품 개요

### 1.1 제품명
**PhotoPick** (포토픽)

### 1.2 제품 설명
PhotoPick은 사진 촬영 스튜디오를 소개하고 예약을 도와주는 플랫폼입니다. 사용자는 지역, 촬영 유형, 가격대 등 다양한 조건으로 스튜디오를 검색하고, 실시간으로 가용 일정을 확인하여 예약할 수 있습니다.

### 1.3 목표 사용자
- **일반 사용자**: 프로필, 가족, 웨딩, 증명사진 등 촬영이 필요한 개인
- **스튜디오 운영자**: 촬영 서비스를 제공하는 스튜디오 사업자
- **기업 고객**: 단체 촬영, 상업 촬영이 필요한 기업

### 1.4 핵심 가치
- 다양한 스튜디오를 한눈에 비교
- 실시간 예약 가능 일정 확인
- 포트폴리오 기반 스튜디오 선택
- 투명한 가격 정보 제공
- 리뷰 기반 신뢰도 확보

## 5. 기술 스택

### 5.1 Frontend
- **Framework**: Next.js 14 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS + CSS Variables
- **State Management**: Zustand
- **Form**: React Hook Form + Zod
- **Animation**: Framer Motion
- **Slider**: Swiper
- **Icons**: Lucide Icons
- **Date**: date-fns
- **Map**: Kakao Maps API

### 5.2 Backend
- **Runtime**: Node.js
- **Framework**: Next.js API Routes 또는 별도 Express/Fastify
- **Database**: PostgreSQL
- **ORM**: Prisma
- **Authentication**: NextAuth.js
- **File Storage**: AWS S3 또는 Cloudflare R2
- **Payment**: Toss Payments / Kakao Pay

### 5.3 Infrastructure
- **Hosting**: Vercel 또는 AWS
- **CDN**: Cloudflare
- **Monitoring**: Sentry
- **Analytics**: Google Analytics 4
